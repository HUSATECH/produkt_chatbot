# Assets Ordner

## Husky Logo

Bitte lege das Husky-Maskottchen-Bild hier ab als:
- **Dateiname:** `husky-logo.png`
- **Format:** PNG oder WebP (empfohlen: PNG mit transparentem Hintergrund)
- **Größe:** Empfohlen 120x120px oder größer (wird automatisch auf 60x60px skaliert)

Das Bild wird im Header als Logo verwendet.
